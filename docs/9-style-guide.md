## 1. 디자인 컨셉 (Design Concept)

본 가이드는 제공된 이미지의 **머티리얼 디자인(Material Design)** 스타일을 계승합니다. 깨끗한 화이트 배경, 명확한 그리드 시스템, 그리고 컬러 칩을 활용한 정보 분류가 핵심입니다.

## 4. 컴포넌트 가이드 (Component Guide)

### 4.1 이벤트 칩 (Event Chips)
- **형태**: 모서리가 약간 둥근 직사각형 (Border-radius: 4px)
- **패딩**: 상하 2px, 좌우 8px
- **효과**: 겹치는 일정의 경우 반투명도 조절 또는 너비 조정

### 4.2 만들기 버튼 (FAB Style)
- **형태**: 둥근 모서리가 큰 화이트 버튼 (Rounded-full)
- **그림자**: `box-shadow: 0 1px 3px 0 rgba(60,64,67,0.30), 0 4px 8px 3px rgba(60,64,67,0.15)`
- **구성**: 다채로운 색상의 `+` 아이콘과 "만들기" 텍스트

### 4.3 체크박스 (Checkboxes)
- **스타일**: 커스텀 컬러 지원 체크박스
- **상태**: 선택 시 해당 카테고리 색상으로 채워지며 체크 표시

---

## 5. 레이아웃 및 그리드 (Layout & Spacing)

- **사이드바 너비**: 약 256px
- **그리드 간격**: 1시간 단위의 일정한 높이 유지
- **거터 (Gutter)**: 그리드 라인 두께 1px (`#E0E0E0`)
- **현재 시간 선**: 빨간색(#EA4335) 수평선과 원형 포인트로 표시

---

## 6. 인터랙션 가이드

- **Hover**: 그리드 셀 호버 시 배경색 미세하게 변경 (`#F1F3F4`)
- **Drag & Drop**: 일정 이동 시 칩의 투명도를 조절하여 드래그 상태 표시
- **Click**: 일정 클릭 시 상세 정보 팝오버(Popover) 노출

---

## 7. 구현 시 주의사항

1. **반응형**: 화면 너비에 따라 요일 표시 형식을 변경합니다. (예: 월요일 -> 월 -> M)
2. **접근성**: 이벤트 칩 배경색에 따라 텍스트 색상을 자동으로 Black 또는 White로 전환하는 로직이 필요합니다.
3. **일관성**: 모든 아이콘은 가느다란 선 위주의 미니멀한 스타일을 유지합니다.
